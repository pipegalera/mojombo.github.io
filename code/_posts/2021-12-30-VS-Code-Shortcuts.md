# {{ page.title }}

- Fullscreen: `F11`
- Zen mode: `ctrl + K -> Z`
- Markdown Preview: `ctrl + K -> V` or `Ctrl+Shift+V`
- Terminal: `ctrl + '` or `ctrl + Shift + Y`
- Command Palette (search): `ctrl + Shift + P`
- Find & Replace: `ctrl + H`
